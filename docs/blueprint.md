# **App Name**: Clubhouse AI

## Core Features:

- AI Command Center: AI-powered command tool to manage club tasks via natural language prompts, including generating announcements, managing member lists, creating social media content, and more.
- Integrated Calendar: Calendar view for all club events with detailed information and RSVP functionality.
- RevTrak Integration: Secure integration with RevTrak for managing financial transactions, dues, and fundraising activities (mock data in MVP).
- Member Directory: Directory of all club members with contact information and messaging capabilities.
- Announcement Channel: Announcement channel for distributing messages to all members or subgroups.
- AI Meeting Slides Generator: AI-assisted tool to automatically generate and format meeting slides based on club data and president input.
- Social Media Manager: AI tool to create and schedule social media posts. Can also use phone camera tool to take and compile images to create posts and publish on social media.

## Style Guidelines:

- Primary color: Light purple (#B19CD9) for a modern, approachable feel.
- Background color: Very light purple (#F4F0F8), almost white, for a clean, unobtrusive backdrop.
- Accent color: Light Blue (#ADD8E6) to complement the primary color and highlight interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) for a blend of modern and warmth, suitable for both headlines and body text.
- Use simple, line-based icons to represent different club functions and events.
- Clean and modern layout with a focus on intuitive navigation.
- Subtle transitions and animations for a smooth user experience.
